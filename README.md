# SIMPLES CALCULADORA

Este Projecto Tem como Objectivo implementar as operações basicas de uma calculadora (Adição , Subtração, Multiplicação, Divisão,Potencia e Raiz Quadrada), Para uma calculadora versão web.

## FUNCIONALIDADES

<ol>
    <li>Adição</li>
    <li>Subtração</li>
    <li>Multiplicação</li>
    <li>Divisão</li>
    <li>Potenciação</li>
</ol>
